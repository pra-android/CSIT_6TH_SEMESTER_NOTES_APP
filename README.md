# CSIT_6TH_SEMESTER_NOTESAPP
It is an offline app which provides materials and notes for CSIT 6th semester. It cane be install on both android and IOS.

## Getting Started
 1. git clone githubrepolink
 2. cd CSIT6thsem_Notesapp
 3. flutter pub get
 4. flutter run
